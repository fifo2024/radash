## 基础用法

给定一个对象数组和一个用于确定每个对象如何被识别的身份回调函数。该函数返回一个对象，其中键是回调函数返回的ID值，每个值是一个整数，表示该ID出现的次数。

```ts
import { counting } from 'radash'

const gods = [
  {
    name: 'Ra',
    culture: 'egypt'
  },
  {
    name: 'Zeus',
    culture: 'greek'
  },
  {
    name: 'Loki',
    culture: 'greek'
  }
]

counting(gods, g => g.culture) // => { egypt: 1, greek: 2 }
```
