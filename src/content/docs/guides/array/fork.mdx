## 基础用法

给定一个项目数组和一个条件，返回两个数组，其中第一个数组包含所有通过条件的项目，第二个数组包含所有未通过条件的项目。

```ts
import { fork } from 'radash'

const gods = [
  {
    name: 'Ra',
    power: 100
  },
  {
    name: 'Zeus',
    power: 98
  },
  {
    name: 'Loki',
    power: 72
  },
  {
    name: 'Vishnu',
    power: 100
  }
]

const [finalGods, lesserGods] = fork(gods, f => f.power > 90) // [[ra, vishnu, zues], [loki]]
```
