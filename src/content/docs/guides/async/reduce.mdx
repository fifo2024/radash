## 基础用法

一个支持回调函数返回 Promise 的归约（Reduce）实现

```ts
import { reduce } from "radash";

const userIds = [1, 2, 3, 4];

const users = await reduce(
    userIds,
    async (acc, userId) => {
        const user = await api.users.find(userId);
        return {
            ...acc,
            [userId]: user,
        };
    },
    {}
);
```
