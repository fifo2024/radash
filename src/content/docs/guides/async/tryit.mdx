## 基础用法

tryit 函数提供了一种优雅的方式将同步/异步函数转换为 error-first 回调风格，同时避免了传统方案中通过可变变量提升状态的问题。

```ts
import { tryit } from "radash";

const [err, user] = await tryit(api.users.find)(userId);
```

### 柯里化 (Currying)

以下是 柯里化版 tryit 的实现，既支持直接调用，也支持分步传递参数：

```ts
import { tryit } from "radash";

const findUser = tryit(api.users.find);

const [err, user] = await findUser(userId);
```
